# Hot Raspberry Sauce

Hackity hack.

Hacking together a home automation system that will (one day) control my heating system.

All it does right now is turn an LED in my living room on and off via a web app.

Mucking about with:
- Raspberry Pi
- Node.js
- Express
- Jade
- Stylus
- Passport
